WWM WebFonts

WWM WebFonts is a collection of Myanmar or ethnics web safe unicode fonts. It is easy and quick to use for everyone with a line of code.
It's supports for Windows, Linux, Mac OSX and iOS.

Just add css external link url in your website. WWM WebFonts is a opensource and you can fork on github(100% free).
